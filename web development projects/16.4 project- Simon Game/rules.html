Rules of the Game

  • 1. Press any key to start the game.
  • 2. Watch the sequence of colors displayed.
  • 3. Repeat the sequence by clicking the colored buttons in the same order.
  • 4. If you successfully repeat the sequence, the next level will start with an additional color in the sequence.
  • 5. If you make a mistake, the game will end, and you can restart by pressing any key.
